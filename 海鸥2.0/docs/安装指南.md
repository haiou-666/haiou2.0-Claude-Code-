# Seagull 安装指南

## 快速安装

### Windows
1. 下载并解压
2. 双击 `启动.bat`
3. 重启 Claude Code
4. 输入 `在吗` 测试

### macOS
1. 下载并解压
2. 双击 `启动.command`（首次需要右键→打开）
3. 重启 Claude Code
4. 输入 `在吗` 测试

> macOS 首次运行 .command 文件可能提示"无法验证开发者"，右键→打开即可。

## 系统要求

### Windows
- Windows 10/11
- PowerShell 5.1+
- Claude Code 已安装

### macOS
- macOS 12+ (Monterey 或更新)
- Claude Code 已安装
- Terminal.app 或 iTerm2

## 安装步骤详解

### 第一步：下载和解压

1. 从发布页面下载 `打开c语言.zip`
2. 右键选择"解压到当前文件夹"
3. 得到 `打开c语言` 文件夹

### 第二步：运行部署

1. 进入 `打开c语言` 文件夹
2. 双击 `启动.bat`
3. 等待部署完成
4. 看到 "Deploy complete!" 表示成功

### 第三步：验证安装

1. 双击 `验证.bat`
2. 看到 "All checks passed!" 表示安装正确
3. 或运行 `快速测试.bat` 进行快速检查

### 第四步：测试功能

1. 打开 Claude Code
2. 输入 `在吗` 或 `海鸥`
3. 应该看到："海鸥在线，你要整点薯条吗？"
4. 继续对话测试性格

## 常见问题

### Q: 中文乱码怎么办？
A: Windows确保系统区域设置支持UTF-8。macOS一般无此问题。

### Q: PowerShell 执行策略错误？（Windows）
A: 使用启动.bat（自动处理），或手动运行：
```powershell
powershell -ExecutionPolicy Bypass -File deploy.ps1
```

### Q: macOS 提示"无法打开"？
A: 右键 `.command` 文件 → 打开。或在终端运行：
```bash
chmod +x 启动.command && ./启动.command
```

### Q: macOS 提示"无法验证开发者"？
A: 系统偏好设置 → 安全性与隐私 → 仍然允许。或终端运行：
```bash
xattr -d com.apple.quarantine 启动.command
```

### Q: 权限不足？
A: Windows以管理员身份运行。macOS检查文件权限：
```bash
chmod +x seagull-files/install.sh
```

### Q: Claude Code 不响应问候？
A: 
1. 重启 Claude Code
2. 运行验证脚本检查配置
3. 确认 config.toml 存在

### Q: 如何卸载？
A: 
- Windows: 双击 `卸载.bat`
- macOS: 双击 `卸载.command`

### Q: 如何恢复备份？
A: 
- Windows: 双击 `恢复备份.bat`
- macOS: 手动从 `~/.claude/backups/` 恢复

## 备份位置

所有备份保存在：
```
# Windows
%USERPROFILE%\.claude\backups\seagull-*

# macOS
~/.claude/backups/seagull-*
```

## 命令行使用

### Windows
```powershell
# 部署
.\seagull-files\deploy.ps1

# 卸载
.\seagull-files\deploy.ps1 -Uninstall

# 验证
.\seagull-files\deploy.ps1 -Verify

# 恢复备份
.\seagull-files\deploy.ps1 -Restore
```

### macOS
```bash
# 部署
bash seagull-files/install.sh

# 卸载
bash scripts/uninstall.sh

# 验证
bash scripts/verify.sh
```

## 高级配置

### 修改问候语

编辑 `seagull-files/claude-config-bundle/CLAUDE.md`，找到问候部分修改。

### 添加术语

编辑 `seagull-files/claude-config-bundle/system-prompt.md`，在术语映射部分添加。

### 修改权限

编辑 `seagull-files/deploy.ps1`，修改 settings.json 部分。
