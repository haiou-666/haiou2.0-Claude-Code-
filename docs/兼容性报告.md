## 已知问题

- PowerShell 2.0 可能不支持某些高级功能
- 非 ASCII 路径在某些旧系统上可能有问题
- 长路径 (>240) 在旧版 Windows 上可能失败

## 建议

- 使用 Windows 10+ 和 PowerShell 5.1+ 获得最佳体验
- 以管理员身份运行可解决大部分权限问题
- 遇到问题时运行 `scripts\兼容性测试.bat` 诊断
