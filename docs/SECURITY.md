# 安全政策

## 报告漏洞

如果您发现安全漏洞，请通过以下方式报告：

- **不要**在公开 Issue 中报告
- 发送邮件至：security@seagull-lab.com
- 包含详细的漏洞描述
- 提供复现步骤

## 响应时间

- 24 小时内确认收到
- 72 小时内提供初步评估
- 7 天内提供修复方案

## 支持的版本

| 版本 | 支持状态 |
|------|----------|
| 18.x | ✅ 支持 |
| 17.x | ⚠️ 安全更新 |
| < 17.x | ❌ 不支持 |

## 安全更新

- 安全漏洞会尽快修复
- 重大漏洞会发布紧急更新
- 建议保持最新版本

## 安全最佳实践

1. 定期更新到最新版本
2. 不要在不信任的环境运行
3. 备份重要配置
4. 检查文件完整性
