# 行为准则

## 我们的承诺

为了营造一个开放、友好、包容的环境，我们作为贡献者和维护者承诺：

- 尊重所有人
- 接受建设性批评
- 关注对社区最有利的事情
- 对他人表示同情

## 我们的标准

### 积极行为

- 使用友好、包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表示同情

### 不当行为

- 使用性暗示的语言或图像
- 恶意评论、人身攻击或政治攻击
- 公开或私下骚扰
- 未经许可发布他人的私人信息
- 其他不适当的行为

## 执行

如果发现不当行为，请通过以下方式报告：

- GitHub Issues
- 邮件：conduct@seagull-lab.com

所有投诉都会被审查和调查，并会做出必要和适当的回应。
